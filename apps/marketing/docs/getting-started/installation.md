# Installation
To install Alette Signal, run the following command
in your terminal:
::: code-group
```sh [pnpm]
pnpm add @alette/signal@latest
```

```sh [npm]
npm install @alette/signal@latest
```

```sh [yarn]
yarn add @alette/signal@latest
```
:::
